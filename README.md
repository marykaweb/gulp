## ✨ Возможности

✔️ Gulp 4  
✔️ SCSS (Sass)  
✔️ Bootstrap 4.6.2  
✔️ jQuery + Popper.js  
✔️ Babel (ES6 → ES5)  
✔️ Autoprefixer  
✔️ Минификация CSS / JS  
✔️ Оптимизация изображений  
✔️ WebP  
✔️ SVG-спрайты  
✔️ HTML include (`@@include`)  
✔️ BrowserSync (live reload)

## 🚀 Как запустить?

### Установка зависимостей
```bash
npm install
```

### Режим разработки
```bash
npm run dev
```

### Продакшен
```bash
npm run build
```
